# Ames-Housing-Capstone-Project
Machine Learning Project predicting house prices in Ames, Iowa, from an Excel database.
First, you should pip install the requirements.txt file to gain all the needed libraries, as well as create the two files listed below.
data:
  The data file should store the CSV file.
model:
  This will store the model once trained.

Make sure to train the model before attempting to run the app.
